# Kaloso App - Base Next.js

Base para convertir Cursos Kaloso en plataforma.

## Incluye
- Landing principal
- Login visual
- Área de alumnos visual
- Panel admin visual
- API placeholder para Mercado Pago
- API placeholder para webhook
- Estructura preparada para Supabase

## Cómo correr localmente
1. Instalar Node.js.
2. Abrir esta carpeta en VS Code.
3. Ejecutar:

```bash
npm install
npm run dev
```

4. Abrir http://localhost:3000

## Próximo paso
Conectar Supabase y Mercado Pago.
